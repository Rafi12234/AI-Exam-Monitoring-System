## 🎯 API ENDPOINTS

```
GET    /api/flagged_students           Get all flagged students
GET    /api/students/<id>/offenses     Get offenses for student
POST   /api/log_offense                Log a new offense
POST   /api/reset_student/<id>         Reset a student
POST   /api/clear_offenses/<id>        Clear offenses
GET    /api/stats                      Get statistics
GET    /health                         Health check
```

## 🐛 TROUBLESHOOTING

### Backend won't start
```bash
# Check Python is installed
python --version

# Check port 5000 is available
# Check dependencies installed
pip install -r requirements.txt
```

### Frontend won't start
```bash
# Check Node.js is installed
node --version

# Check npm packages installed
npm install

# Check port 3000 is available
```

### Camera/CV issues
- Check camera permissions
- Verify backend is running
- Check `cv_pipeline.py` video_source setting

### WebSocket connection error
- Ensure backend is running on port 5000
- Check browser console for errors
- Clear browser cache
